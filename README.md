# FelinoTweets [![Build Status](https://travis-ci.org/FelinoSoft/FelinoTweets.svg?branch=master)](https://travis-ci.org/FelinoSoft/FelinoTweets)

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

Aplicación web para gestionar múltiples cuentas de Twitter desde un único lugar de una manera fácil y sencilla.

NOTA IMPORTANTE: los Tokens para acceder al API de Twitter han sido cambiadas, por lo tanto, la aplicación no funcionará (será necesario añadir unos Tokens de Twitter válidos).
